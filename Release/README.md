Chimera Complex

How to Play:

PC-------------------
Movement:   Arrow Keys
Shoot:      Left Ctrl
Open Door:  Space

Controller-----------
Movement:   Left Stick
Camera:     Right Stick
Shoot:      Right Bumper/R1
Open Door:  Left Bumper/L1
